# Контекст: Задачи (tasks)

**Фокус:** Work/Задачи/, планы на день, переносы, АКТУАЛЬНЫЕ_ЗАДАЧИ, синхронизация с ClickUp.

## Что соблюдать

- **task-management.mdc** — проверка даты, перенос на новый день, счётчики [→N], лимит 5–7 активных.
- **actual-tasks-sync.mdc** — при любом изменении плана/задач обновлять `АКТУАЛЬНЫЕ_ЗАДАЧИ.md`.
- **Work/Задачи/.cursorrules** — тон общения (диалог, без игрофикации), структура плана, запреты.

## Ключевые файлы

- `Work/Задачи/План на ДД.ММ.ГГГГ.md` — план на день.
- `Work/Задачи/Задачи.md` — долгосрочный список.
- `Work/Задачи/АКТУАЛЬНЫЕ_ЗАДАЧИ.md` — обзор (автообновление).
- `Work/Задачи/Когда-нибудь.md` — бэклог.
- `Work/Задачи/scripts/new_day.ps1` — перенос на новый день.

## При начале сессии (в этом контексте)

1. Проверить текущую дату (user_info).
2. Найти последний план. Если дата плана < сегодня — предложить перенос.
3. Не раздувать ответы; предлагать следующий шаг.

## ClickUp

При наличии — следовать `.cursor/rules/clickup-sync.mdc` и сохранять **ClickUp ID:** при переносе задач.
