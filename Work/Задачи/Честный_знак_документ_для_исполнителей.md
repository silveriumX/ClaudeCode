# Честный знак (маркировка): документ для исполнителей

**Назначение:** Единый список вопросов и проблем по системе «Честный знак» (Wildberries, ФБС). Исполнители, изучая этот документ, сразу заполняют сроки, способы решения, статус (решили / не решили) и всю информацию в одном месте.

**Контекст:** Ревизия по Честному знаку, фулфилмент, бухгалтерия. Нужно привести в порядок кабинет ЧЗ, вывести из оборота проданные КИЗ по ФБС, устранить ошибки по отгрузкам без маркировки или с неправильными кодами.

**Связанные файлы:** `Честный_знак_задание_на_фриланс.md` (задание для фриланса по выводу КИЗ), `Задачи.md` (задача 43).

---

## Как пользоваться документом

1. **Ответственный** — при назначении на проблему впиши своё имя и срок.
2. **Способ решения** — опиши, что именно делаешь (шаги, ЛК ЧЗ/WB, документы, консультации).
3. **Статус** — обновляй по ходу: `не начато` → `в работе` → `решено` или `не решено (причина)`.
4. **Результат** — кратко: что сделано, какой вывод, ссылка на инструкцию или «не решено из‑за …».
5. **Дата обновления** — ставь дату при каждом изменении строки.
