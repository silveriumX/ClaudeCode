# Задачи на 06.02 — что делать, если самому ничего не делать (только рабочие)

Для каждой **рабочей** задачи: что делегировать, кому и как, чтобы день прошёл без твоей исполнительской работы.

Заполни первыми 5–10 пунктами из наших чатов/задач». Дальше поддержку документа — на них. |
| **Честный знак: вывод КИЗ из оборота** — Дима HR, фриланс, инфо | Уже делегировано Диме. Твоё действие ноль: только чек-поинт вечером «Дима, какой статус по КИЗ?» и при необходимости «пришли выводы в 2 предложениях». |

## Главное на сегодня

### Максим и делегирование

| Задача | Если ничего сам не делать |
|--------|---------------------------|
| **Максим — оплата** (неделя, тесты) | Максиму: «Пришли расчёт: сколько за неделю + тесты, в каком формате удобно (разово/по этапам)». Или бухгалтеру/Наташе: «Согласуй с Максимом оплату за прошлую неделю и тесты, потом скажи итог». |
| **Максим — структура** (научить, взять задачи, разгрузить) | Максиму + Нигиар: «Вот регламент/чат/документ со структурой. С сегодняшнего дня ты ведёшь замены и кейсы по этому регламенту; Нигиар — по Proxima и доступам. Ко мне эскалируйте только то, что не решается по регламенту». Обучение — один раз показал/запись — дальше они по инструкции. |
| **Нигиар — Proxima** (инструкции) | Максим или тот, кто уже умеет: «Запиши инструкцию по Proxima (или дай доступ к существующей), передай Нигиар и один раз пройди с ней по шагам». Ты только проверяешь, что инструкция есть и доступ передан. |
| **Нигиар — доступы** (Ads Power, все сервера, смена портов) | Максиму: «Выдай Нигиар Ads Power и доступ ко всем серверам по регламенту. Настрой так, чтобы вы с ней могли менять порты в течение дня без меня». Ты только утверждаешь регламент доступа. |
| **Регламент** (замены/починка) | Максиму: «Сделай регламент: кто что делает при падении/замене, куда пишем, кто меняет порты. До вечера черновик в документ». Ты только правки по смыслу, не писание с нуля. |
| **Нигиар — видео** (первые замены) | Нигиар: «При первых 2–3 заменах запиши экран/себя и сохрани в общую папку. Это будет инструкция для следующих». Контроль — Максим: «Проверь, что Нигиар записала». |
| **Максим — скриншоты по кейсам** (база решений) | Максиму: «По каждому кейсу присылай скриншот + кратко: что было, что сделали. Складываем в [папка/чат/нотка]. К концу недели — краткая база: при таком-то — делаем так». Ты только смотришь базу при выборе решения. |

## Итог

- Почти все **рабочие** задачи можно закрыть без твоей исполнительской работы: чёткое поручение + ответственный + формат отчёта («что сделано / что прислать тебе»).
- Единственное, что остаётся за тобой: **решения** (согласовать оплату Максиму, выбрать юриста/бухгалтера, сказать «да/нет» по выкупам, утвердить регламент и т.п.) и при необходимости **одноразовое** обучение/передача (если никто ещё не умеет).
- Документы на сегодня (читать и учить), ЭКГ, забрать вещи, здоровье, ПДД, Япония, красота, Даша, одежда — не рабочие; их сюда не включал.
