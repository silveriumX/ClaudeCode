# Карта полного делегирования: «Что если вообще ничего не делать самому»

**Дата:** 02.02.2026
**Цель:** Разобрать все задачи из плана и списков. Ответить на вопрос: «Как бы ты подошёл к решению, если бы хотел ничего вообще не делать из списка задач?»

---

## Принцип делегирования

Есть 4 типа задач:

1. **Только ты** — решения, которые нельзя передать (утверждение стратегии, финальный выбор).
2. **Ты + помощник** — ты формулируешь запрос, кто-то собирает информацию/делает черновик, ты утверждаешь.
3. **Полностью делегируется** — достаточно поставить задачу, человек сам доводит до конца и отчитывается.
4. **Автоматизируется** — можно сделать скрипт/бот/систему, чтобы вообще не трогать руками.

## ИТОГО: Схема «Ничего не делать самому»

### Шаг 1: Нанять ключевых людей

1. **Личный ассистент (Асель на полную ставку или +помощник)**
   - Основная зона: коммуникации, организация, выплаты, мониторинг задач, бытовые вопросы.
   - Доступы: календарь, почта, чаты, кошельки (под контролем), таблицы.

2. **HR-менеджер (Дима или +1 человек)**
   - Основная зона: найм специалистов (юристы, бухгалтеры, технари, фрилансеры).
   - Результат: ты получаешь готовых кандидатов или уже работающих исполнителей.

3. **Технический специалист (Максим-сисадмин + Станислав по автоматизации)**
   - Основная зона: серверы, боты, автоматизация процессов, скрипты.
   - Результат: технические задачи решаются без твоего участия, ты только утверждаешь архитектуру.

4. **Менеджер по закупкам (можно Асель + обучение или отдельный человек)**
   - Основная зона: закупки (Китай, поставщики, брокеры), отслеживание партий.
   - Результат: ты утверждаешь крупные суммы, всё остальное ведёт менеджер.

5. **Бухгалтер(ы) для каждого ИП (+ главный бухгалтер-координатор)**
   - Основная зона: налоги, отчёты, сверки, платежи по налогам.
   - Результат: ты не трогаешь бухгалтерию вообще, только получаешь отчёты.

### Шаг 2: Создать систему делегирования

1. **Все входящие задачи → фильтр:**
   - «Только я могу решить» → в твой список (должно быть 1–3 задачи в день).
   - «Нужна моя экспертиза» → ты + помощник (ты формулируешь/утверждаешь, помощник исполняет).
   - «Можно полностью делегировать» → сразу кому-то из команды (Асель, Дима, Надя, Станислав, Максим).

2. **Стандартные процессы → шаблоны и чек-листы:**
   - Например: «Найти специалиста» → есть шаблон для Димы HR (где искать, как отбирать, как согласовывать).
   - «Провести выплаты» → есть чек-лист для Асель (открыть таблицу, провести платежи, скриншоты, отчёт).

3. **Регулярные задачи → автоматизация или расписание:**
   - Проверка налоговой → Асель каждую пятницу.
   - Выплаты зарплат → Асель в установленные даты.
   - Отчёты от исполнителей (Максим, Борис, Станислав) → они присылают сами раз в 2–3 дня.

### Шаг 3: Твоя роль — только решения и утверждения

Ты занимаешься только:

1. **Стратегические решения:** на какой магазин переносить карточки, запускать ли выкупы, в какую страну регистрировать компанию, с каким брокером работать.
2. **Утверждения:** финальный выбор из 2–3 вариантов, которые тебе подготовили (кандидаты, бухгалтер, название компании, план действий).
3. **Контроль:** получение кратких отчётов (1 раз в день или через день) от ключевых людей (Асель, Дима, Максим, бухгалтер). Формат отчёта: «что сделано, что в процессе, что требует твоего решения».

Всё остальное делают другие.
