# Инструкция по добавлению колонки "Ссылка на чек"

## Что нужно сделать

Добавить новую колонку "Ссылка на чек" в конец каждого листа с заявками.

---

## Шаг 1: Лист "Основные"

1. Откройте лист **"Основные"**
2. Найдите последнюю заполненную колонку (должна быть **S: "Автор Полное имя"**)
3. В ячейку **T1** вставьте заголовок: `Ссылка на чек`
4. Готово!

**Результат:** теперь у листа 20 колонок (A-T)

---

## Шаг 2: Лист "USDT"

1. Откройте лист **"USDT"**
2. Найдите последнюю заполненную колонку (должна быть **M: "Автор Полное имя"**)
3. В ячейку **N1** вставьте заголовок: `Ссылка на чек`
4. Готово!

**Результат:** теперь у листа 14 колонок (A-N)

---

## Шаг 3: Лист "Разные выплаты" (опционально)

Этот лист используется только для чтения старых заявок. Если у вас есть оплаченные заявки на этом листе, к которым нужно прикрепить чеки:

1. Откройте лист **"Разные выплаты"**
2. Найдите последнюю колонку
3. Добавьте заголовок `Ссылка на чек` в следующую колонку
4. Готово!

---

## Шаг 4: Лист "USDT Зарплаты" (опционально)

Этот лист используется только для чтения старых заявок. Если у вас есть оплаченные заявки на этом листе, к которым нужно прикрепить чеки:

1. Откройте лист **"USDT Зарплаты"**
2. Найдите последнюю колонку
3. Добавьте заголовок `Ссылка на чек` в следующую колонку
4. Готово!

---

## Шаг 5: Создайте папку на Google Drive (если еще не создана)

1. Откройте [Google Drive](https://drive.google.com)
2. Создайте новую папку с названием **"Чеки Finance Bot"**
3. Настройте доступ:
   - Нажмите правой кнопкой на папку → "Поделиться"
   - Добавьте всех исполнителей (или сделайте доступ по ссылке)
   - Установите права: **"Редактор"** (чтобы могли загружать файлы)
4. Скопируйте ссылку на папку для инструкции

---

## Проверка

После добавления колонок:

1. Убедитесь, что заголовок написан правильно: `Ссылка на чек`
2. Проверьте, что колонка в правильном месте (последняя после "Автор Полное имя")
3. Попробуйте вставить тестовую ссылку в любую строку с оплаченной заявкой
4. Откройте бота → "Мои заявки" → выберите эту заявку
5. Убедитесь, что ссылка отображается

---

## Формат ссылки

Правильная ссылка на файл в Google Drive выглядит так:

```
https://drive.google.com/file/d/1AbCdEfGhIjKlMnOpQrStUvWxYz/view
```

Или так (сокращённая):

```
https://drive.google.com/file/d/1AbCdEfGhIjKlMnOpQrStUvWxYz
```

**Неправильные форматы:**
- ❌ Прямая ссылка на скачивание (`/export?format=...`)
- ❌ Ссылка на папку вместо файла
- ❌ Пустая ссылка или текст без `https://`

---

## Что дальше?

После настройки таблицы бот автоматически начнёт отображать ссылки на чеки в карточках оплаченных заявок.

Исполнителям нужно будет:
1. Загружать чеки в папку "Чеки Finance Bot"
2. Копировать ссылку на файл
3. Вставлять ссылку в колонку "Ссылка на чек" в таблице

Подробная инструкция для исполнителей: `RECEIPTS_GUIDE.md`
