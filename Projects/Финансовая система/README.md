# Финансовая система

Система автоматизации личных финансов и учета через Google Sheets.

## Описание

Система предоставляет:
- Автоматический учет доходов и расходов
- Интеграция с банковскими выписками
- Категоризация транзакций
- Финансовые отчеты и аналитика
- Планирование бюджета
- Отслеживание целей накоплений

## Структура

```
Финансовая система/
├── Копия Фин. отчет (Общий).xlsx  # Основной отчет (13MB)
└── Скрипты автоматизации
```

## Использование

1. Настроить доступ к Google Sheets API
2. Создать копию шаблона таблицы
3. Настроить автоматический импорт транзакций
4. Система автоматически обновляет отчеты

## Интеграции

- Google Sheets API - основное хранилище
- Банковские API - автоматический импорт
- Telegram бот - быстрый ввод расходов

## Требования

- google-api-python-client
- gspread
- pandas (для аналитики)

## Конфигурация

См. корневой `.env` для настройки Google Sheets credentials.
