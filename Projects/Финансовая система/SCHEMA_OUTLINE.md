## 2. Юрлица
- DBZ
- MN
- VAS

## 3. Бизнес-банк
- Расчётные счета (1:1 с юрлицами): DBZ, MN, VAS, LYA, MAKS, LIFE, ALEX, HUB
- Оплата по безналу
- Налоги

## 4. Личные карты
- Личные карты физлица (у каждого юрлица свои)
- Маркет-карта (СБП на карту)

## 5. Варианты с карт
- Наличные (снятие по юрлицам)
- Сейф на складе (наличные → обмен через трейдера, офис → USDT)
- P2P Обмен (карта → USDT; Bybit пополняется через P2P переводы)
- Оплата сервисов
- Зарплата ИП

## 6. USDT-кошельки
- USDT Зарплаты (холодный)
- RUB/BYN Зарплаты (холодный)
- Разные выплаты (холодный)
- Аргентина (холодный)
- Китай/Карго (холодный)
- Холодный транзит (Bybit → Карго/HTX)
- Кошелёк Карго

## 7. Сервисы
- P2P Платформы (BitPapa; пополнение с холодного USDT)
- Bybit (пополнение через P2P переводы; пока только DBZ)
- HTX (юани Alipay, Китай)

## 8. Получатели
- Сотрудники (оплаты с P2P)
- Поставщики (Китай, Карго)
- Прочие (дизайнеры, курьеры, сервисы)
