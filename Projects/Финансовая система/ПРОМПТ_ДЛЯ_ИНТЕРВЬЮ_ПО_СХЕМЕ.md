# Промпт для агента: интервью по схеме потоков

Скопируй текст ниже в новый чат и прикрепи файл `СХЕМА_ПОТОКОВ_ОПИСАНИЕ.md` (или весь проект/папку «Финансовая система»).

---

**Промпт:**

Работаем с описанием финансовой схемы потоков. Файл @СХЕМА_ПОТОКОВ_ОПИСАНИЕ.md — единственный источник истины: в нём текстовое описание блоков 1–8, узлов и связей. Визуализация (CLEAN_FLOWCHART.html) строится только после моего подтверждения.

**Твоя задача:** пройтись по каждому блоку (1 → 2 → … → 8) в формате структурированного интервью и довести текстовое описание схемы до полной и однозначной готовности. Результат — обновлённый СХЕМА_ПОТОКОВ_ОПИСАНИЕ.md, по которому потом можно однозначно собрать HTML-схему.

**Правила работы:**

1. **Только текст.** Все правки вноси только в СХЕМА_ПОТОКОВ_ОПИСАНИЕ.md. Файл CLEAN_FLOWCHART.html не меняй, пока я явно не скажу: «описание готово для визуализации» или «можно делать HTML».

2. **По одному блоку за раз.** Начинай с блока 1. Для каждого блока:
   - кратко зачитай текущее описание блока (узлы, связи);
   - задай 2–5 уточняющих вопросов по: названиям, количеству узлов, откуда/куда идут связи, исключениям, подписям;
   - после моих ответов сразу внеси правки в .md и покажи обновлённый фрагмент;
   - спроси: «По блоку N так верно? Переходим к блоку N+1?» — и только после подтверждения переходи к следующему блоку.

3. **Вопросы задавай конкретно.** Не «всё ли верно?», а: «Магазинов ровно 9 или нужно добавить/убрать?», «Ozone только в Your Leads или ещё в какие магазины идёт поток?», «Р/С ровно 9, по одному на магазин?» и т.п.

4. **Связи проверяй явно.** Для каждого блока уточняй: какие узлы откуда и куда связаны, есть ли связи «все к одному» / «один ко всем», обратные потоки, двойные пути (например, р/с → карта напрямую и р/с → Market-карта → СБП → карта).

5. **Фиксируй неясности.** Если я говорю «пока не знаю» или «уточним потом» — добавь в раздел «Открытые вопросы» в .md и не додумывай за меня.

6. **В конце дай сводку.** После блока 8: кратко перечисли, что изменилось по блокам, и напомни: «Когда скажешь, что описание полностью готово для визуализации — обновлю CLEAN_FLOWCHART.html по этому описанию.»

Начни с блока 1 (Маркетплейсы): зачитай его описание из файла и задай первые уточняющие вопросы.
