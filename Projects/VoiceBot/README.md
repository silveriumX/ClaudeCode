# VoiceBot

Telegram бот для работы с голосовыми сообщениями и распознавания речи.

## Описание

Бот предоставляет:
- Распознавание голосовых сообщений в текст (Speech-to-Text)
- Синтез речи из текста (Text-to-Speech)
- Перевод голосовых между языками
- Сохранение транскриптов

## Использование

1. Настроить `.env` с API ключами
2. Запустить: `python bot.py`
3. Отправить голосовое сообщение боту

## Команды

- `/start` - Начать работу
- `/help` - Помощь
- `/language` - Выбрать язык распознавания
- Отправить голосовое - получить текст

## Требования

- python-telegram-bot
- speech_recognition / OpenAI Whisper
- pydub (для обработки аудио)

## Конфигурация

См. `.env` для настройки API ключей распознавания речи.
