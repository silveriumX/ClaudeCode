# Настройка бота для записи мыслей

## Шаг 1: Создать Telegram бота

1. Открой @BotFather в Telegram
2. Отправь `/newbot`
3. Введи имя бота (например: "Мои мысли")
4. Введи username (например: `my_thoughts_12345_bot`)
5. Скопируй токен (выглядит как `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

## Шаг 2: Узнать свой Telegram ID

1. Открой @userinfobot в Telegram
2. Нажми Start
3. Скопируй свой ID (число)

## Шаг 3: Получить OpenAI API ключ

1. Зайди на https://platform.openai.com/api-keys
2. Создай новый ключ
3. Скопируй (начинается с `sk-...`)

## Шаг 4: Настроить бота

Открой `bot.py` и замени:

```python
TELEGRAM_TOKEN = "ТВОЙ_TELEGRAM_BOT_TOKEN"      # из шага 1
OPENAI_API_KEY = "ТВОЙ_OPENAI_API_KEY"          # из шага 3
ALLOWED_USER_ID = 123456789                      # из шага 2
```

## Шаг 5: Установить зависимости

```bash
cd C:\Users\Admin\Documents\Cursor\VoiceBot
pip install -r requirements.txt
```

## Шаг 6: Запустить

```bash
python bot.py
```

## Использование

1. Открой своего бота в Telegram
2. Записывай голосовые — они транскрибируются и сохраняются
3. Можно отправлять и текст
4. Файлы появляются в папке `C:\Users\Admin\Documents\Cursor\Мысли\`

### Команды
- `/start` — приветствие
- `/today` — показать мысли за сегодня
- `/clear` — очистить сегодняшний файл

## Чтобы бот работал постоянно

Вариант 1: Оставить терминал открытым

Вариант 2: Запустить как службу Windows (Task Scheduler)

Вариант 3: Захостить на сервере (Railway, Render, VPS)
