# Сейвер

Telegram бот для автоматического сохранения контента из чатов.

## Описание

Бот автоматически сохраняет:
- Сообщения из указанных чатов
- Медиа файлы (фото, видео, документы)
- Пересланные сообщения
- Ссылки и важный контент

Полезен для:
- Архивирования важных обсуждений
- Сохранения контента из временных чатов
- Создания личной базы знаний

## Использование

1. Настроить `.env` с токеном бота
2. Добавить бота в нужные чаты
3. Запустить: `python bot.py`
4. Бот автоматически сохраняет контент

## Команды

- `/start` - Начать работу
- `/add_chat` - Добавить чат для мониторинга
- `/list_chats` - Список отслеживаемых чатов
- `/export` - Экспорт сохраненного контента

## Требования

- python-telegram-bot
- База данных для хранения (SQLite/PostgreSQL)

## Конфигурация

См. `.env` для настройки токена и параметров сохранения.
