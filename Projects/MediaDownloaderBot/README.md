# MediaDownloaderBot

Telegram бот для скачивания медиа из различных источников.

## Описание

Бот позволяет скачивать:
- Видео с YouTube, TikTok, Instagram
- Аудио треки
- Изображения
- Истории из соцсетей

## Использование

1. Настроить `.env` с токеном бота
2. Запустить: `python bot.py`
3. Отправить боту ссылку на медиа

## Команды

- `/start` - Начать работу
- `/help` - Помощь
- Просто отправь ссылку - бот скачает медиа

## Требования

- python-telegram-bot
- yt-dlp (для YouTube)
- requests

См. `.env.example` для примера настройки.
