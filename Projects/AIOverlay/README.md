# AI Overlay

AI-ассистент для написания ответов в Telegram. Работает с AyuGram, Telegram Desktop и другими клиентами.

## Возможности

- **3 варианта ответа** - разные подходы и формулировки
- **Автообучение стилю** - AI учится твоему стилю письма
- **Свободный ввод** - генерация текста без контекста чата
- **Кастомные инструкции** - настрой tone of voice под себя
- **Всегда поверх окон** - удобно при работе

## Быстрый старт

### 1. Установи зависимости

```bash
pip install -r requirements.txt
```

### 2. Настрой API ключ

Скопируй `config.example.txt` в `config.txt` и вставь свой Claude API ключ:

```
API_KEY: sk-ant-api03-xxxxx
MODEL: claude-sonnet-4-5
```

Получить ключ: https://console.anthropic.com/

### 3. Запусти

```bash
python ai_overlay.py
```

Или двойной клик на `START.bat`

## Использование

### Режим 1: Ответ на сообщение (F9)

1. Выдели текст в чате
2. Скопируй (Ctrl+C)
3. Нажми **F9**
4. Выбери вариант и скопируй

### Режим 2: Свободный ввод (F8)

1. Нажми **F8**
2. Введи запрос в поле
3. Нажми Enter
4. Выбери вариант и скопируй

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| **F9** | Ответ на скопированный текст |
| **F8** | Свободный ввод (без контекста) |
| **1 / 2 / 3** | Переключить вариант |
| **Enter** | Перегенерировать |
| **Ctrl+Enter** | Скопировать и закрыть |
| **Esc** | Закрыть |

## Файлы

```
AIOverlay/

├── config.txt             # Твой API ключ (создай из example)
├── config.example.txt     # Пример конфига
├── custom_instructions.txt # Правила стиля (редактируй!)
├── requirements.txt       # Зависимости Python
├── START.bat              # Запуск на Windows
├── build_exe.bat          # Сборка .exe

└── learned_style.txt      # Выученный стиль (авто)
```

## Настройка стиля

Редактируй `custom_instructions.txt`:

```
MY_NAMES: Имя1, Имя2

## Стиль письма
- НЕ использовать длинное тире (—)
- Писать кратко и по делу
- Адаптироваться под собеседника

## Запрещённые слова
- НЕ использовать "сорри"
- НЕ использовать эмодзи
```

## Сборка .exe

Для передачи коллегам:

```bash
build_exe.bat
```

Готовый файл: `dist/AI_Overlay.exe`

Для работы нужен `config.txt` рядом с .exe
