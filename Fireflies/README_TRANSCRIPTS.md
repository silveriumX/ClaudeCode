# Полные транскрипты встреч Fireflies

В каждом файле встречи (`2026-01-*.md`) есть:
- **Краткое содержание** (саммари от Fireflies)
- **Ключевые слова**
- **Action items**
- **Полный транскрипт** — диалог по репликам (добавлен в конец файла)

Если в `.md` транскрипт обрезан из‑за размера, полный текст доступен:
1. По ссылке **Transcript Url** в блоке метаданных внизу каждого файла (формат Fireflies).
2. Через API: `fireflies_get_transcript(transcriptId)` — ID указан в начале каждого файла.

Транскрипты получены через Fireflies API 31.01.2026.
